# System Pracowniczy

Prosta aplikacja GUI zbudowana w Pythonie, służąca do rejestracji czasu pracy.

## Funkcjonalności
- Logowanie i rejestracja (dane zapisywane w JSON)
- Stoper mierzący czas pracy
- Interaktywny interfejs (zmiana kolorów przycisków)

## Jak uruchomić
Wymagany Python 3.x.
Uruchom plik główny:
`RegLog.py`
